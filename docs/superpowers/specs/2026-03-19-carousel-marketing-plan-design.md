# BalloonBase — Carousel Marketing Plan

**Date:** 2026-03-19
**Platforms:** TikTok, Instagram
**Format:** 5-carousel launch series
**Strategy:** Pain → Solution arc

---

## Overview

A 5-carousel launch series for BalloonBase targeting balloon decorators and artists who run their own businesses. The series uses a Pain → Solution arc — each carousel opens with a relatable pain point the audience faces daily, agitates it, then reveals BalloonBase as the fix. Every carousel closes with a sign-up CTA.

**Goals:** Brand awareness + sign-ups + community building
**Tone:** Professional, fun/relatable, and always educational

---

- **Colors:** Orange (#F26B00) + Cream/Beige
- **Typography:** Bubbly/rounded (matches logo)
- **Logo:** Inflated 3D "B" icon on cream rounded square + "Balloon Base" wordmark

## Asset Specifications

### Slide Dimensions
- **Instagram carousels:** 1080×1350px (4:5 portrait)
- **TikTok slideshows:** 1080×1920px (9:16 vertical) — resize Instagram slides, increase font size by ~20%, keep text out of top/bottom 15% (UI chrome zones)

### App Screenshots Needed
| Carousel | Slide | Screen to Capture |
|----------|-------|-------------------|
| 1 | 4 | Dashboard — customer list view with sample job data |
| 2 | 4 | Quote builder — catalog items added, total visible |
| 3 | 4 | Length estimator — filled with sample design input + output quantities |
| 4 | 4 | Booking form + calendar side by side, or two sequential screens |

> For Carousel 3 (Length Estimator): use a before/after format — Slide 3 shows a scribbled manual calculation (use a mockup of handwritten notes on paper, or a photo of a notepad); Slide 4 shows the BalloonBase estimator UI with real input values and bold output quantities. The output numbers should be UI-native (as they appear in the app) — no text overlays needed if they're already readable.

> **Screenshot note:** All app screenshots should use realistic but anonymised sample data (e.g., client names like "Sarah J." not real names, job descriptions like "Birthday arch — 5ft"). Dashboard screenshot should show 4–6 jobs to feel active without being cluttered.

### Brand Application
- **Primary:** Orange (#F26B00) on cream (#FFF8F0) backgrounds
- **Text overlays:** White text on orange blocks; dark (#1A1A1A) text on cream
- **Font:** Nunito (primary). Minimum 32px body text for mobile readability
- **Logo placement:** Bottom-right corner of every solution and CTA slide

## Posting Strategy

- **Sequence:** Post in order (1 → 5)
- **Cadence:** Every 3 days — 5 posts over 13 days total
- **Schedule:** Post 1 (Day 1) → Post 2 (Day 4) → Post 3 (Day 7) → Post 4 (Day 10) → Post 5 (Day 13)
- **Best posting times:** Tue–Fri, 7–9pm local time (peak engagement for service business owners)

### Conversion Tracking
- **Action required:** Set up a link-in-bio tool (Linktree or equivalent) pointing to your BalloonBase sign-up page before Day 1
- Tag each carousel separately with UTM parameters: `?utm_source=instagram&utm_campaign=carousel1` (increment campaign number per post)
- Use the same UTM pattern for TikTok with `utm_source=tiktok`
- This is required before the first post goes live — otherwise sign-up attribution is impossible

## Caption Formula

Write captions to stand alone — someone who only reads the caption (not the slides) should still get the full message.

1. **Open with a bold statement** — a stronger version of the hook (not a repeat of Slide 1)
2. **Expand the pain** (2–3 short sentences, conversational, first-person "we" or "you")
3. **Name the solution** (1 sentence — don't over-explain, let the slides do it)
4. **CTA** — "Try BalloonBase free → link in bio" (use your tracked link)

> Example for Carousel 3: "Balloon artists waste hundreds every year over-ordering — or worse, run short mid-job. Nobody teaches you how to calculate quantities when you're starting out. BalloonBase has a free length estimator built in. Link in bio 🎈 #balloonbusiness"

## Success Metrics

### What to Track Per Carousel
- **Saves** — primary signal; above 2% save rate = strong content
- **Shares** — secondary signal; indicates community resonance
- **Profile visits** — awareness signal
- **Link-in-bio clicks** — conversion signal (use UTM per carousel)
- **New sign-ups** — ultimate conversion metric

### Rough Benchmarks (early stage account)
- Saves ≥ 2% of reach = good
- Link clicks ≥ 0.5% of reach = acceptable
- Use Carousel 3 (Length Estimator) performance as baseline — it has the highest uniqueness; if it underperforms the others, revisit the hook

### Optimisation Rule
After all 5 are posted, identify which carousel had the highest save rate. Double down on that pain point in the next content series.
